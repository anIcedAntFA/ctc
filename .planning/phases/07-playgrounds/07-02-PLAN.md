# Phase 7 Plan 02: playground/react

## Goal

Scaffold `playground/react` as a Vite + React 19 app demonstrating `useCopyToClipboard` from `@ngockhoi96/ctc-react` with full UX contract (copy button, copied state, error display, secure context badge, detection panel).

## Context

- **D-01**: Located at `playground/react/` (per D-01).
- **D-03**: Detection panel shows all 4 detect functions from `@ngockhoi96/ctc`, evaluated on load.
- **D-05**: Independent styling — React playground uses its own CSS (no shared design tokens).
- **D-06**: Every playground must demonstrate: copy button, "Copied!" 2s feedback, error code display, secure context badge, detection panel.
- **D-07**: `"private": true`, CI-excluded.
- **D-08**: `"dev": "vite"` script in `package.json`.
- **RESEARCH**: Use `@vitejs/plugin-react@6.0.1` (Babel, not SWC). React 19 (`^19.2.5`) in playground — latest stable, plugin-react@6 has no React version peer dep. `@types/react@^19.2.14` and `@types/react-dom@^19.2.3` must be in playground devDeps (not inherited from adapter package). Vite 8.0.8 pinned.

- **Wave 2**: Depends on Plan 01 (workspace config + pnpm install ran). `playground/*` is already registered in `pnpm-workspace.yaml`.

2. Create `playground/react/tsconfig.json`:

   {

6. Create `playground/react/src/App.tsx` — full demo (D-06, D-03). Uses `useCopyToClipboard` from `@ngockhoi96/ctc-react`. Detection panel calls the 4 detect functions from `@ngockhoi96/ctc` on mount via `useEffect`. The 2s reset timer is local state managed inside `App` (not exported from adapter — per D-06 note):
   ```typescript
   import {
     isClipboardApiSupported,
     isReadPermissionGranted,
     isSecureContext,
     isWritePermissionGranted,
   } from '@ngockhoi96/ctc'
   import { useCopyToClipboard } from '@ngockhoi96/ctc-react'
   import { useEffect, useState } from 'react'

   type DetectionState = {
     clipboardApi: boolean | null
     secureContext: boolean | null
     readPermission: boolean | null
     writePermission: boolean | null
   }

   export function App() {
     const [text, setText] = useState('Hello from @ngockhoi96/ctc-react')
     const { copy, copied, error } = useCopyToClipboard()
     const [detection, setDetection] = useState<DetectionState>({
       clipboardApi: null,
       secureContext: null,
       readPermission: null,
       writePermission: null,
     })

     useEffect(() => {
       const clipboardApi = isClipboardApiSupported()
       const secureContext = isSecureContext()
       setDetection((prev) => ({ ...prev, clipboardApi, secureContext }))

       Promise.all([isReadPermissionGranted(), isWritePermissionGranted()]).then(
         ([readPermission, writePermission]) => {
           setDetection((prev) => ({ ...prev, readPermission, writePermission }))
         },
       )
     }, [])

     const secure = detection.secureContext

     return (
       <div style={{ fontFamily: 'system-ui, sans-serif', maxWidth: 600, margin: '2rem auto', padding: '0 1rem' }}>
         <h1 style={{ fontSize: '1.25rem', marginBottom: '1.5rem' }}>
           @ngockhoi96/ctc -- React Playground
         </h1>

         <section style={{ marginBottom: '1.5rem' }}>
           <h2 style={{ fontSize: '1rem', marginBottom: '0.5rem' }}>Copy to Clipboard</h2>
           <input
             type="text"
             value={text}
             onChange={(e) => setText(e.target.value)}
             style={{ border: '1px solid #ccc', padding: '0.4rem 0.6rem', borderRadius: 4, width: '100%', boxSizing: 'border-box', marginBottom: '0.5rem' }}
           />
           <button
             onClick={() => copy(text)}
             style={{ padding: '0.4rem 1rem', borderRadius: 4, border: '1px solid #888', cursor: 'pointer', background: '#f4f4f4' }}
           >
             Copy
           </button>
           <div style={{ marginTop: '0.4rem', fontSize: '0.875rem', minHeight: '1.25em' }}>
             {copied && <span style={{ color: 'green' }}>Copied!</span>}
             {!copied && error !== null && <span style={{ color: 'red' }}>Error: {error}</span>}
           </div>
         </section>

         <section style={{ marginBottom: '1.5rem' }}>
           <h2 style={{ fontSize: '1rem', marginBottom: '0.5rem' }}>Secure Context</h2>
           {secure !== null && (
             <span style={{
               display: 'inline-block',
               padding: '0.2rem 0.5rem',
               borderRadius: 4,
               fontSize: '0.75rem',
               fontWeight: 'bold',
               background: secure ? '#d4edda' : '#f8d7da',
               color: secure ? '#155724' : '#721c24',
             }}>
               {secure ? 'Secure Context' : 'Insecure Context'}
             </span>
           )}
         </section>

         <section>
           <h2 style={{ fontSize: '1rem', marginBottom: '0.5rem' }}>Detection Panel</h2>
           <table style={{ borderCollapse: 'collapse', width: '100%', fontSize: '0.875rem' }}>
             <thead>
               <tr>
                 <th style={{ textAlign: 'left', padding: '0.3rem 0.5rem', background: '#f9f9f9', borderBottom: '1px solid #eee' }}>Function</th>
                 <th style={{ textAlign: 'left', padding: '0.3rem 0.5rem', background: '#f9f9f9', borderBottom: '1px solid #eee' }}>Result</th>
               </tr>
             </thead>
             <tbody>
               {[
                 { label: 'isClipboardApiSupported()', value: detection.clipboardApi },
                 { label: 'isSecureContext()', value: detection.secureContext },
                 { label: 'isReadPermissionGranted()', value: detection.readPermission },
                 { label: 'isWritePermissionGranted()', value: detection.writePermission },
               ].map(({ label, value }) => (
                 <tr key={label}>
                   <td style={{ padding: '0.3rem 0.5rem', borderBottom: '1px solid #eee' }}>{label}</td>
                   <td style={{ padding: '0.3rem 0.5rem', borderBottom: '1px solid #eee' }}>
                     {value === null ? 'checking...' : String(value)}
                   </td>
                 </tr>
               ))}
             </tbody>
           </table>
         </section>
       </div>
     )
   }
   ```

## Notes

- **React 19 in playground, React 18 in adapter**: `packages/react` was built against React 18.3.1. The React playground uses React 19. This is intentional — the adapter has no React version peer dep ceiling, and React 19 is the current stable. The playground is a greenfield consumer.
- **Pitfall 6 (workspace:* version resolution)**: `@ngockhoi96/ctc-react` must have a `version` field in its `package.json` before `pnpm install`. Check before running install.
- **useCopyToClipboard error field**: The adapter returns `{ copy, copied, error }` where `error` is a `BrowserUtilsError` code string or `null`. Display it directly — do not throw.
- **2s timer is local**: The adapter's `useCopyToClipboard` handles the `copied` reset internally (2s default). No additional timer is needed in `App.tsx`.
