# Phase 7 Plan 03: playground/vue

## Goal

Scaffold `playground/vue` as a Vite + Vue 3 app demonstrating `useCopyToClipboard` from `@ngockhoi96/ctc-vue` with full UX contract (copy button, copied state, error display, secure context badge, detection panel).

## Context

- **D-01**: Located at `playground/vue/` (per D-01).
- **D-03**: Detection panel shows all 4 detect functions from `@ngockhoi96/ctc`, evaluated on load.
- **D-05**: Independent styling — Vue playground uses its own scoped styles inside `App.vue`.
- **D-06**: Every playground must demonstrate: copy button, "Copied!" 2s feedback, error code display, secure context badge, detection panel.
- **D-07**: `"private": true`, CI-excluded.
- **D-08**: `"dev": "vite"` script in `package.json`.
- **RESEARCH**: Use `@vitejs/plugin-vue@6.0.6` (latest, supports Vite 8). Vue 3.5.32 to match installed version in `packages/vue/node_modules/vue`. Vue 3 ships its own types — no `@types/vue` needed.
- **RESEARCH Pitfall 5**: TypeScript cannot resolve `.vue` file imports without a type declaration. Add `playground/vue/src/env.d.ts` with `/// <reference types="vite/client" />` to register `.vue` module types.
- **RESEARCH Pitfall 6**: `@ngockhoi96/ctc-vue` must have a `version` field in its `package.json` for pnpm `workspace:*` to resolve.
- **Wave 2**: Depends on Plan 01. Runs in parallel with Plan 02 (no file overlap).

7. Create `playground/vue/src/App.vue` — full demo using Vue 3 Composition API (D-06, D-03). Uses `useCopyToClipboard` from `@ngockhoi96/ctc-vue`. The 2s reset is handled internally by the adapter. Detection panel uses `onMounted` + `Promise.all` for async checks:
   ```vue
   <script setup lang="ts">
   import {
     isClipboardApiSupported,
     isReadPermissionGranted,
     isSecureContext,
     isWritePermissionGranted,
   } from '@ngockhoi96/ctc'
   import { useCopyToClipboard } from '@ngockhoi96/ctc-vue'
   import { onMounted, ref } from 'vue'

   const text = ref('Hello from @ngockhoi96/ctc-vue')
   const { copy, copied, error } = useCopyToClipboard()

   type Detection = {
     clipboardApi: boolean | null
     secureContext: boolean | null
     readPermission: boolean | null
     writePermission: boolean | null
   }

   const detection = ref<Detection>({
     clipboardApi: null,
     secureContext: null,
     readPermission: null,
     writePermission: null,
   })

   onMounted(async () => {
     detection.value.clipboardApi = isClipboardApiSupported()
     detection.value.secureContext = isSecureContext()
     const [read, write] = await Promise.all([
       isReadPermissionGranted(),
       isWritePermissionGranted(),
     ])
     detection.value.readPermission = read
     detection.value.writePermission = write
   })

   const detectionRows = [
     { key: 'clipboardApi' as const, label: 'isClipboardApiSupported()' },
     { key: 'secureContext' as const, label: 'isSecureContext()' },
     { key: 'readPermission' as const, label: 'isReadPermissionGranted()' },
     { key: 'writePermission' as const, label: 'isWritePermissionGranted()' },
   ]
   </script>

   <template>
     <div class="container">
       <h1>@ngockhoi96/ctc -- Vue Playground</h1>

       <section>
         <h2>Copy to Clipboard</h2>
         <input v-model="text" type="text" />
         <button @click="copy(text)">Copy</button>
         <div class="feedback">
           <span v-if="copied" class="copied">Copied!</span>
           <span v-else-if="error !== null" class="error">Error: {{ error }}</span>
         </div>
       </section>

       <section>
         <h2>Secure Context</h2>
         <span
           v-if="detection.secureContext !== null"
           class="badge"
           :class="detection.secureContext ? 'secure' : 'insecure'"
         >
           {{ detection.secureContext ? 'Secure Context' : 'Insecure Context' }}
         </span>
       </section>

       <section>
         <h2>Detection Panel</h2>
         <table>
           <thead>
             <tr>
               <th>Function</th>
               <th>Result</th>
             </tr>
           </thead>
           <tbody>
             <tr v-for="row in detectionRows" :key="row.key">
               <td>{{ row.label }}</td>
               <td>{{ detection[row.key] === null ? 'checking...' : String(detection[row.key]) }}</td>
             </tr>
           </tbody>
         </table>
       </section>
     </div>
   </template>

   <style scoped>
   .container {
     font-family: system-ui, sans-serif;
     max-width: 600px;
     margin: 2rem auto;
     padding: 0 1rem;
   }
   h1 { font-size: 1.25rem; margin-bottom: 1.5rem; }
   section { margin-bottom: 1.5rem; }
   h2 { font-size: 1rem; margin-bottom: 0.5rem; }
   input {
     border: 1px solid #ccc;
     padding: 0.4rem 0.6rem;
     border-radius: 4px;
     width: 100%;
     box-sizing: border-box;
     margin-bottom: 0.5rem;
   }
   button {
     padding: 0.4rem 1rem;
     border-radius: 4px;
     border: 1px solid #888;
     cursor: pointer;
     background: #f4f4f4;
   }
   button:hover { background: #e0e0e0; }
   .feedback { margin-top: 0.4rem; font-size: 0.875rem; min-height: 1.25em; }
   .copied { color: green; }
   .error { color: red; }
   .badge {
     display: inline-block;
     padding: 0.2rem 0.5rem;
     border-radius: 4px;
     font-size: 0.75rem;
     font-weight: bold;
   }
   .secure { background: #d4edda; color: #155724; }
   .insecure { background: #f8d7da; color: #721c24; }
   table { border-collapse: collapse; width: 100%; font-size: 0.875rem; }
   th, td { text-align: left; padding: 0.3rem 0.5rem; border-bottom: 1px solid #eee; }
   th { background: #f9f9f9; }
   </style>
   ```

## Notes

- **Pitfall 5 (vue module types):** Without `env.d.ts`, TypeScript errors on `import App from './App.vue'`. The `/// <reference types="vite/client" />` line in `env.d.ts` registers Vite's `.vue` module declaration.
- **Pitfall 6 (workspace:* version):** `@ngockhoi96/ctc-vue` must have a `version` field in its `package.json`. Check before running `pnpm install`.
- **Biome + .vue files:** Biome does not lint `.vue` files (not in its includes, and Vue SFC format is not TypeScript). Only `main.ts` and `env.d.ts` in `playground/vue/src/` are linted. Vite handles `.vue` compilation.
- **useCopyToClipboard from ctc-vue returns refs:** `copy` is a function, `copied` and `error` are Vue `Ref<boolean>` and `Ref<string | null>`. Use `copied` and `error` directly in the template — Vue's template auto-unwraps refs.
- **Wave 2 parallelism:** Plans 02, 03, and 04 are all in Wave 2 — they have no file overlap and can run in parallel or sequentially.
