## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| DOM element -> Action | Click event triggers clipboard write via copyRichAction |
| Component -> Core | Runes/stores variants delegate to core copyRichContent |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-05 | T (Tampering) | copyRichAction params | accept | Caller-provided HTML/text passed to core; clipboard API handles write security |
