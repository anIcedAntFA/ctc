Guard sequence in copy.ts / read.ts:
1. isBrowser() → false → return false (CLIPBOARD_NOT_SUPPORTED)
2. isSecureContext() → false → return false (INSECURE_CONTEXT)
3. typeof navigator.clipboard?.writeText !== 'function' → true → return false (CLIPBOARD_NOT_SUPPORTED)
4. navigator.clipboard.writeText throws NotAllowedError → return false (CLIPBOARD_PERMISSION_DENIED)
5. navigator.clipboard.writeText throws Error → return false (CLIPBOARD_WRITE_FAILED)
6. success → return true

Guard sequence in fallback.ts:
1. isBrowser() → false → return false (CLIPBOARD_NOT_SUPPORTED)
2. !document.body → return false (CLIPBOARD_NOT_SUPPORTED)
3. document.execCommand('copy') returns false → return false (CLIPBOARD_WRITE_FAILED)
4. execCommand throws → return false (CLIPBOARD_WRITE_FAILED)
5. textarea.isConnected=false in finally → skip remove()
6. success → return true

handleError branches:
1. onError provided → call it (and swallow if it throws)
2. onError absent + expected error code → console.warn
3. onError absent + unexpected error code → console.error
