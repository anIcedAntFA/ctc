API REFERENCE SECTION — per D-14 each function needs: TypeScript signature, one working example, return value description, browser support note:

Document all 5 functions:
1. `copyToClipboard(text, options?)` → Promise<boolean>
   - Return: `true` on success, `false` on any failure (never throws)
   - Browser support: Requires HTTPS (secure context) and a user gesture. Chrome 66+, Firefox 63+, Safari 13.1+.
   - Example: button click handler copying text

2. `readFromClipboard(options?)` → Promise<string | null>
   - Return: clipboard text string on success, `null` on failure
   - Browser support: Requires HTTPS and user gesture. May trigger permission prompt on first call. Chrome 66+, Firefox 63+, Safari 13.1+.
   - Example: paste button reading clipboard

3. `isClipboardSupported()` → boolean
   - Return: `true` if clipboard write is available in the current context
   - Browser support: Synchronous, no permissions needed. Returns false on HTTP and in SSR.
   - Example: conditional UI showing copy button vs fallback

4. `isClipboardReadSupported()` → boolean
   - Return: `true` if clipboard read is available in the current context
   - Browser support: Same as isClipboardSupported but for read API.
   - Example: conditional paste button

5. `copyToClipboardLegacy(text, options?)` → boolean (synchronous)
   - Return: `true` on success, `false` on failure
   - Browser support: Works on HTTP pages via deprecated execCommand. Not reliable on iOS Safari. Use only when modern API is unavailable.
   - Example: HTTP page fallback

ERROR HANDLING SECTION:
Document BrowserUtilsError interface and all 5 error codes with descriptions:
- CLIPBOARD_NOT_SUPPORTED: API unavailable (SSR, very old browser)
- INSECURE_CONTEXT: Page is HTTP (not HTTPS). Use copyToClipboardLegacy() instead.
- CLIPBOARD_PERMISSION_DENIED: User denied clipboard permission, or call was outside user gesture
- CLIPBOARD_WRITE_FAILED: Write operation failed for unexpected reason
- CLIPBOARD_READ_FAILED: Read operation failed for unexpected reason

Include onError example:
```typescript
await copyToClipboard('text', {
  onError: (err) => {
    if (err.code === 'CLIPBOARD_PERMISSION_DENIED') {
      showPermissionPrompt()
    }
  }
})
```
