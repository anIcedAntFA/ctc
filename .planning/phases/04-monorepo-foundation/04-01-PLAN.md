# Phase 4, Plan 1: Workspace Scaffold + Core Migration

**Phase:** 04-monorepo-foundation
**Plan:** 1 of 2
**Requirements:** MONO-01, MONO-02, MONO-03, MONO-04
**Estimated tasks:** 4

## Goal

Create the monorepo skeleton and physically migrate the core library into `packages/core/`. This plan installs the structural foundation — pnpm workspace config, Turborepo pipeline, shared TypeScript base config — then moves all source, test, and build-tool files from the repo root into `packages/core/` with no code changes. After this plan completes, `packages/core` is a self-contained, buildable, testable package that publishes as `@ngockhoi96/ctc` exactly as before.

## Pre-conditions

- Repo is on a feature branch (never commit to master directly — per CLAUDE.md)
- Root `package.json` is the current single-package manifest (version 0.2.1)
- `src/`, `tests/`, `tsdown.config.ts`, `vitest.config.ts`, `playwright.config.ts` exist at repo root
- `pnpm install` succeeds from root with the current lockfile
- There are no uncommitted changes
